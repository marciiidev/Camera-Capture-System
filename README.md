# 📸 Camera Capture System

A web-based camera capture system built using **HTML, CSS, and JavaScript** that allows users to take photos directly from the browser using webcam access. The system uses the browser’s built-in camera API to capture and save images instantly with a simple and responsive interface.

---

## 🚀 Features

- 📷 Access webcam directly from browser  
- 🖼️ Capture images in real-time  
- 💾 Download captured photos  
- 📱 Fully responsive design (mobile & desktop)  
- ⚡ Fast and lightweight frontend system  
- 🧑‍💻 Simple and user-friendly interface  

---

## 🛠️ Technologies Used

- HTML5 (Structure)
- CSS3 (Design)
- JavaScript (WebRTC / getUserMedia API)

## ⚙️ How It Works

1. Open the project in a browser  
2. Allow camera permission  
3. The webcam stream starts automatically  
4. Click **Capture** to take a photo  
5. Download or save the image
